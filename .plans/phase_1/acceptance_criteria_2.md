# 模块 2: 快捷键编辑器 - 验收标准

## 功能完整性

- [ ] **录制模式**：点击"修改"按钮后进入录制状态
- [ ] **录制模式**：录制状态下监听键盘事件并显示当前按键
- [ ] **快捷键验证**：只接受包含至少一个修饰键的组合（Ctrl/Alt/Shift/Meta）
- [ ] **快捷键验证**：拒绝单独的修饰键或无效组合
- [ ] **保存功能**：点击"保存"后调用后端命令更新快捷键
- [ ] **保存功能**：成功后更新 localStorage 并显示成功提示
- [ ] **取消功能**：点击"取消"退出录制模式且不保存更改
- [ ] **焦点丢失处理**：窗口失去焦点时自动取消录制
