# JSON Explorer 📋

一个现代化的 JSON 查看器和编辑器插件，提供优雅的树形可视化、格式化、路径访问等功能。

## ✨ 功能特性

### 🎨 核心功能
- **实时 JSON 解析** - 粘贴即解析，即时显示错误
- **树形可视化** - 可展开/折叠的交互式 JSON 树
- **智能格式化** - 一键美化 JSON 格式
- **压缩优化** - 移除空白，减小体积
- **快速拷贝** - 支持普通拷贝和转义字符串拷贝
- **JS 路径访问** - 使用 JavaScript 语法访问深层节点（如 `data.users[0].name`）

### 🎯 用户体验
- **现代化 UI** - 渐变背景、毛玻璃效果、流畅动画
- **双面板布局** - 左侧编辑，右侧预览
- **智能提示** - 悬停显示操作按钮
- **状态反馈** - 拷贝成功提示、错误信息显示
- **示例加载** - 一键加载示例 JSON 快速体验

## 🚀 使用方法

### 1. 基本使用
1. 在 Quick Actions 中搜索 "json" 或 "JSON Explorer"
2. 打开插件后，在左侧输入框粘贴 JSON
3. 右侧自动显示树形结构

### 2. 格式化与压缩
- 点击 **Format 🎨** 按钮美化 JSON
- 点击 **Minify 📦** 按钮压缩 JSON
- 点击 **Copy 📋** 拷贝当前 JSON
- 点击 **Copy Escaped 🔐** 拷贝转义后的字符串

### 3. 树形导航
- 点击节点前的 ▶ 箭头展开/折叠
- 悬停在值上显示 **Copy** 按钮快速拷贝
- 根节点显示总键数统计

### 4. JS 路径访问
在底部输入框输入 JavaScript 风格的路径：
```javascript
// 示例路径
data.users[0].name
config.theme
stats.rating
```

点击 **Access & Copy 🎯** 或直接按 Enter：
- 自动解析路径并获取值
- 结果自动拷贝到剪贴板
- 显示成功提示

## 💡 使用场景

### 场景 1: API 响应调试
```json
{
  "status": 200,
  "data": {
    "users": [
      {"id": 1, "name": "Alice"},
      {"id": 2, "name": "Bob"}
    ]
  }
}
```
→ 使用路径 `data.users[0].name` 快速提取 "Alice"

### 场景 2: 配置文件编辑
```json
{
  "theme": "dark",
  "features": {
    "notifications": true,
    "autoSave": false
  }
}
```
→ 格式化后清晰查看配置结构

### 场景 3: JSON 转义
需要将 JSON 作为字符串嵌入代码时：
```javascript
const json = "{\"key\":\"value\"}";  // 使用 Copy Escaped
```

## 🛠️ 技术实现

### 核心技术
- **React 19** - 最新 React 特性
- **TypeScript** - 类型安全
- **Tailwind CSS** - 实用优先的样式
- **Vite** - 快速构建工具

### 关键算法
1. **递归渲染**: 深度优先遍历 JSON 树
2. **路径解析**: 将 `a.b[0].c` 转换为数组 `['a', 'b', '0', 'c']`
3. **状态管理**: 使用 Set 管理展开节点
4. **防抖处理**: 输入时延迟解析避免频繁计算

### 性能优化
- 虚拟滚动（大数据集可扩展）
- 懒加载子节点
- 记忆化计算（useMemo）
