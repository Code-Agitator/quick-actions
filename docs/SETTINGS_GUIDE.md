# 设置页面功能说明

## 概述

设置页面提供了插件管理和系统配置的集中管理界面，用户可以通过搜索框右侧的设置按钮访问。

## 功能模块

### 1. 插件管理 📦

**功能**：
- 查看已安装插件列表
- 启用/禁用插件
- 查看插件版本和描述
- 安装新插件（待实现）

**当前插件**：
- Everything 搜索 - 通过 Everything 搜索本地文件
- 计算器 - 快速计算数学表达式
- 命令运行器 - 执行系统命令
- 文件浏览器 - 浏览和管理文件系统

### 2. 外观设置 🎨

**主题模式**：
- 跟随系统 - 自动适配系统主题
- 浅色模式 - 强制使用浅色主题
- 深色模式 - 强制使用深色主题

**主题预览**：
- 提供三种主题的可视化预览
- 点击即可切换主题

### 3. 通用设置 ⚙️

**启动设置**：
- ✅ 开机自启 - 系统启动时自动运行
- ✅ 显示托盘图标 - 在系统托盘显示应用图标

**语言与区域**：
- 简体中文
- English

**动画效果**：
- ✅ 启用动画 - 开启界面过渡动画效果

### 4. 关于 ℹ️

**应用信息**：
- 应用名称：Quick Actions
- 版本号：0.1.0
- 技术栈：Tauri + React + TypeScript + Rust

**相关链接**：
- GitHub 仓库
- 技术栈说明

## 界面设计

### 布局结构

```
┌─────────────────────────────────────┐
│  设置标题                    [关闭]  │  <- Header
├─────────────────────────────────────┤
│ [插件] [外观] [通用] [关于]         │  <- Tabs
├─────────────────────────────────────┤
│                                     │
│         内容区域                     │
│      (可滚动)                        │
│                                     │
└─────────────────────────────────────┘
```

### 交互设计

1. **Tab 切换**：点击 Tab 标签切换不同设置类别
2. **开关控件**：使用 Toggle Switch 控制布尔值设置
3. **下拉选择**：使用 Select 组件选择枚举值
4. **关闭按钮**：点击右上角关闭按钮或按 ESC 返回主界面

## 技术实现

### 组件结构

```
Settings/
├── PluginsTab      - 插件管理标签页
├── AppearanceTab   - 外观设置标签页
├── GeneralTab      - 通用设置标签页
└── AboutTab        - 关于标签页
```

### 状态管理

当前使用本地 state 管理设置，后续可以：
1. 集成到全局状态管理（如 Zustand、Redux）
2. 持久化到本地存储（localStorage 或 Tauri Store）
3. 同步到后端配置文件

### 样式设计

- 使用 Tailwind CSS 进行样式设计
- 支持深色模式（dark: 前缀）
- 响应式布局
- 平滑过渡动画

## 使用方法

### 打开设置页面

1. 呼出 Quick Actions（Alt+Space）
2. 点击搜索框右侧的 ⚙️ 图标

### 关闭设置页面

1. 点击右上角的 ✕ 按钮
2. 或按 ESC 键

### 修改设置

1. 切换到对应的标签页
2. 修改相应的设置项
3. 设置会自动保存（待实现持久化）

## 注意事项

1. **设置持久化**：目前设置仅在内存中，刷新后会重置
2. **插件管理**：启用/禁用功能需要后端支持
3. **主题切换**：需要集成到全局主题管理系统
4. **语言切换**：需要完整的国际化支持
